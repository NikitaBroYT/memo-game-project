# Итоговый проект по дисциплине Веб-технологии

## ФИО студента, номер учебной группы

Утюпин Никита Денисович, 2141-ДБ

## Тематика игры, название

Игра "Мемо" (найди пару). Идея реализовать игру по правилам настольной игры [Мемо](https://ru.wikipedia.org/wiki/%D0%9C%D0%B5%D0%BC%D0%BE_(%D0%BD%D0%B0%D1%81%D1%82%D0%BE%D0%BB%D1%8C%D0%BD%D0%B0%D1%8F_%D0%B8%D0%B3%D1%80%D0%B0)), с измененными карточками.

## Целевая аудитория

Школьники и студенты, заинтересованные простыми логическими играми.

## Функциональные возможности

* игровое поле с карточками;
* переворот карточки по нажатию ЛКМ;
* одновременное открытие двух карточек;
* фиксация правильно найденной пары;
* возврат неправильно найденных карточек в исходное состояние;
* подсчёт количества ходов;
* подсчёт времени игры на таймере;
* выбор уровня сложности (размер поля)
* кнопка перезапуска игры;
* ...
